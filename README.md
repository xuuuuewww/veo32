# Veo32

这是一个使用 Next.js、TypeScript、Tailwind CSS 和 shadcn/ui 构建的项目。

## 开始使用

首先，安装依赖：

```bash
npm install
```

然后，运行开发服务器：

```bash
npm run dev
```

在浏览器中打开 [http://localhost:3000](http://localhost:3000) 查看结果。

## 技术栈

- **Next.js 14** - React 框架
- **TypeScript** - 类型安全
- **Tailwind CSS** - 样式框架
- **shadcn/ui** - UI 组件库

## 添加 shadcn/ui 组件

使用以下命令添加 shadcn/ui 组件：

```bash
npx shadcn@latest add [component-name]
```

例如：

```bash
npx shadcn@latest add button
npx shadcn@latest add card
```
